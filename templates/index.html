TRS-80 Color Computer BASIC

Try these commands:

  • PRINT "HELLO WORLD"
  • 10 FOR I=1 TO 5: PRINT I: NEXT
  • LIST - Show program
  • RUN - Run program
  • CLEAR - Clear program
  • CLS - Clear screen

Graphics commands:

  • PMODE 4,1: SCREEN 1,1 - Set graphics mode
  • PSET(128,96) - Set pixel
  • LINE(0,0)-(255,191) - Draw line
  • CIRCLE(128,96),50 - Draw circle
  • PCLS - Clear graphics
  • COLOR 4,0 - Set colors (red on black)
  • SOUND 1000,30 - Play tone (1000Hz, 0.5sec)
