Basic Commands

  • PRINT "text" - Display text
  • LIST - Show program
  • RUN - Run program
  • NEW - Clear program
  • SAVE "name" - Save program
  • LOAD "name" - Load program

Graphics Commands

  • PMODE 4,1 - Set graphics mode
  • SCREEN 1,1 - Activate graphics
  • PSET(x,y) - Set pixel
  • LINE(x1,y1)-(x2,y2) - Draw line
  • CIRCLE(x,y),r - Draw circle
  • PAINT(x,y),c,b - Fill area

Dual Monitor Features

  • Drag splitter to resize panels
  • Graphics persist during REPL use
  • Export graphics as PNG/SVG
  • Multiple program tabs
  • Session auto-save
  • Share programs via URL
